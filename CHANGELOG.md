# Changelog

## 0.1.0

Initial release.

- ISF language support for `.fs` and `.vs` files
- JSON header: autocompletion, hover, validation, and formatting
- GLSL body: full IDE support via shadow files delegated to any installed GLSL extension
- ISF version awareness (v1 / v2) with warnings for mismatched built-ins
- Vertex shader support: paired `.vs` files inherit INPUTS and get vertex-specific built-ins
